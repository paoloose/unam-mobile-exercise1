Built using Debian with Android Studio Panda 3

## Extras

- The user is able to change its profile picture by selecting a local image
- The profile data is kept in persistent storage, thanks to the Preferences API
- An "Edit profile" option to open the form in edit mode

## Building

```
./gradlew assembleDebug
```
